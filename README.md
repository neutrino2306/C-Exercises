# C Exercises

This repository contains C exercises I completed while self-learning C programming. It serves as a collection of small practice programs covering language fundamentals, standard library usage, and problem-solving exercises.

## Overview

The files in this repository are mainly organized as standalone exercises, with names based on exercise numbers or topics. They were written as part of my learning process, so the repository reflects iterative practice rather than a single unified project.

## What I Practiced

Through these exercises, I practiced:

- Basic syntax, control flow, and functions
- Arrays, strings, and pointers
- Standard string handling functions such as `strlen`, `strcmp`, and `strcpy`
- Problem decomposition and debugging through small coding tasks
- Writing and testing simple console-based programs

## Repository Notes

- Most source files are individual exercise implementations.
- Some compiled `.exe` files are included alongside the source code.
- File names may follow exercise numbering rather than descriptive project names.

## Purpose

This repository is mainly a record of my early programming practice in C. It helped me build familiarity with core language concepts and develop a stronger foundation in problem solving and implementation.
